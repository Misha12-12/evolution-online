# Evolution Online 🦎

## 📲 Играть прямо сейчас
👉 [Открыть Evolution Online](https://evolution-online.onrender.com)

## 📱 Особенности проекта
- **Полностью в браузере**: ничего не нужно скачивать или устанавливать.
- **Мобильная адаптация**: удобно играть с телефона.
- **Открытый код**: всё написано на чистом HTML/CSS/JS (Vanilla JS), без тяжёлых фреймворков.

## ▶️ Запуск

Игра доступна онлайн: [evolution-online.onrender.com](https://evolution-online.onrender.com).  
Для локальной разработки просто открой `index.html`.

⚠️ **Внимание:** проект находится в разработке. Некоторые функции могут работать нестабильно, а правила игры — меняться.
